Función Throttle

¿Qué es la función throttle?

La función throttle es una herramienta que reduce la frecuencia de ejecución de una función dentro de un intervalo de tiempo.

Para utilizarla, se define una función de callback y un valor de retardo (delay). Durante el delay, la función moderará las llamadas al callback. Un contador registra el número de eventos que ocurren durante este periodo.

El contador de eventos se reinicia cuando el tiempo transcurrido desde la última llamada supera el delay. En este caso el delay propuesto es de 2000ms. El usuario tiene ese tiempo para generar 'clicks' a traves del boton proporcionado. Luego, se muestra el resultado al usuario.
